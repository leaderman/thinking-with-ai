# 微信公众号素材与草稿箱接口调研

## 官方文档地址

- 素材管理：https://developers.weixin.qq.com/doc/offiaccount/Asset_Management/New_temporary_materials.html
- 草稿箱接口：https://developers.weixin.qq.com/doc/offiaccount/Draft_Box/Add_draft.html
- 发布能力接口：https://developers.weixin.qq.com/doc/offiaccount/Draft_Box/Freepublish.html
- 第三方文档参考：https://weixingongzhonghao.com.cn/zh/api/media.html

---

## 一、素材类型说明

| 类型 | 说明 | 有效期 | 是否有公开 URL |
|------|------|------|------|
| 临时素材 | 用于发消息等短期场景 | 3 天 | 无，只能通过 media_id 使用或下载二进制 |
| 永久素材（图片） | 长期存储，上传后返回 URL | 永久（不删除则一直有效） | 有，但仅限在微信生态内使用 |

### 素材格式与大小限制

| 类型 | 格式 | 大小 | 时长 |
|------|------|------|------|
| image | JPG/PNG | ≤2MB | - |
| voice | AMR/MP3 | ≤2MB | ≤60秒 |
| video | MP4 | ≤10MB | - |
| thumb | JPG | ≤64KB | - |

永久素材数量上限：图文/图片各 5000 条，其他类型各 1000 条。

---

## 二、已废弃接口说明

`/cgi-bin/material/add_news`（上传永久图文素材）已于 **2022 年 2 月 28 日**正式废弃，原因是微信将图文素材升级为草稿箱体系。调用该接口会返回：

```json
{ "errcode": 45106, "errmsg": "This API has been unsupported." }
```

**替代方案**：改用草稿箱接口 `/cgi-bin/draft/add`。

## 三、实现功能清单

| 功能 | 接口 | 状态 |
|------|------|------|
| 上传本地图片为临时素材 | `POST /cgi-bin/media/upload` | ✅ 成功 |
| 上传本地图片为永久素材并获取 URL | `POST /cgi-bin/material/add_material` | ✅ 成功 |
| 新建草稿 | `POST /cgi-bin/draft/add` | ✅ 成功 |
| 删除草稿 | `POST /cgi-bin/draft/delete` | ✅ 成功 |
| 发布草稿 | `POST /cgi-bin/freepublish/submit` | ❌ 权限不足（需认证服务号） |

## 四、关键问题与解决方案

### axios 通过代理上传 multipart 返回 503

**原因**：本机设置了 `HTTPS_PROXY=http://127.0.0.1:1234`，axios 会自动读取该环境变量并通过代理发请求，但该代理对 multipart/form-data 的 POST 请求返回 503。curl 能正常工作是因为两者实现差异，代理对 curl 的处理正常。

**解决**：在 axios 请求配置中加 `proxy: false`，绕过代理直连微信 API。

```js
axios.post(url, form, {
  headers: form.getHeaders(),
  proxy: false,
});
```

## 五、代码示例

### 1. 上传临时素材（图片）

**接口**：`POST https://api.weixin.qq.com/cgi-bin/media/upload?access_token=ACCESS_TOKEN&type=image`

**响应**：
```json
{
  "type": "image",
  "media_id": "MEDIA_ID",
  "created_at": 1234567890
}
```

```js
const axios = require('axios');
const FormData = require('form-data');
const fs = require('fs');
const path = require('path');

async function uploadTemporaryMedia(accessToken, filePath) {
  const form = new FormData();
  form.append('media', fs.createReadStream(filePath), {
    filename: path.basename(filePath),
    contentType: 'image/jpeg',
  });

  const url = `https://api.weixin.qq.com/cgi-bin/media/upload?access_token=${accessToken}&type=image`;
  const response = await axios.post(url, form, {
    headers: form.getHeaders(),
    proxy: false,
  });

  return response.data; // { type, media_id, created_at }
}
```

### 2. 上传永久图片素材（获取 URL）

**接口**：`POST https://api.weixin.qq.com/cgi-bin/material/add_material?access_token=ACCESS_TOKEN&type=image`

**响应**：
```json
{
  "media_id": "MEDIA_ID",
  "url": "http://mmbiz.qpic.cn/..."
}
```

> 注意：返回的 `url` 为微信 CDN 地址，仅限在微信图文消息正文中使用，不适合作为通用公开图片链接。

```js
const axios = require('axios');
const FormData = require('form-data');
const fs = require('fs');
const path = require('path');

async function uploadPermanentImage(accessToken, filePath) {
  const form = new FormData();
  form.append('media', fs.createReadStream(filePath), {
    filename: path.basename(filePath),
    contentType: 'image/jpeg',
  });

  const url = `https://api.weixin.qq.com/cgi-bin/material/add_material?access_token=${accessToken}&type=image`;
  const response = await axios.post(url, form, {
    headers: form.getHeaders(),
    proxy: false,
  });

  return response.data; // { media_id, url }
}
```

### 3. 新建草稿

**接口**：`POST https://api.weixin.qq.com/cgi-bin/draft/add?access_token=ACCESS_TOKEN`

**请求参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `title` | string | 是 | 文章标题 |
| `thumb_media_id` | string | 是 | 封面图的永久素材 media_id |
| `author` | string | 否 | 作者 |
| `digest` | string | 否 | 摘要，不填则取正文前 54 字 |
| `show_cover_pic` | number | 否 | 1=显示封面，0=不显示 |
| `content` | string | 是 | 正文 HTML，图片须使用微信域名 |
| `content_source_url` | string | 否 | 原文链接 |
| `need_open_comment` | number | 否 | 1=开启评论，0=关闭 |
| `only_fans_can_comment` | number | 否 | 1=仅粉丝可评论，0=所有人 |

**响应**：
```json
{
  "media_id": "MEDIA_ID",
  "item": [{ "index": 0, "ad_count": 0 }]
}
```

```js
const axios = require('axios');

async function addDraft(accessToken, article) {
  const url = `https://api.weixin.qq.com/cgi-bin/draft/add?access_token=${accessToken}`;
  const response = await axios.post(url, { articles: [article] }, { proxy: false });
  return response.data; // { media_id, item }
}

// 调用示例
addDraft(ACCESS_TOKEN, {
  title: '今天是值得纪念的一天',
  thumb_media_id: 'THUMB_MEDIA_ID',
  author: '闲人一个',
  digest: '这就是一个测试',
  show_cover_pic: 1,
  content: '<p>正文内容...</p><img src="http://mmbiz.qpic.cn/..."/>',
  content_source_url: 'https://example.com',
  need_open_comment: 0,
  only_fans_can_comment: 0,
});
```

### 4. 删除草稿

**接口**：`POST https://api.weixin.qq.com/cgi-bin/draft/delete?access_token=ACCESS_TOKEN`

**请求参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `media_id` | string | 是 | 草稿的 media_id |

**响应**：
```json
{ "errcode": 0, "errmsg": "ok" }
```

```js
const axios = require('axios');

async function deleteDraft(accessToken, mediaId) {
  const url = `https://api.weixin.qq.com/cgi-bin/draft/delete?access_token=${accessToken}`;
  const response = await axios.post(url, { media_id: mediaId }, { proxy: false });
  return response.data;
}
```

### 5. 发布草稿（需认证服务号权限）

**接口**：`POST https://api.weixin.qq.com/cgi-bin/freepublish/submit?access_token=ACCESS_TOKEN`

> 注意：该接口需要公众号已开通发布能力，通常仅认证服务号可用。订阅号或未认证账号调用会返回 `errcode: 48001`（接口未授权）。

**第一步：提交发布**

```js
async function submitPublish(accessToken, mediaId) {
  const url = `https://api.weixin.qq.com/cgi-bin/freepublish/submit?access_token=${accessToken}`;
  const response = await axios.post(url, { media_id: mediaId }, { proxy: false });
  return response.data; // { errcode, errmsg, publish_id }
}
```

**第二步：查询发布状态**

`POST https://api.weixin.qq.com/cgi-bin/freepublish/querystatus?access_token=ACCESS_TOKEN`

`publish_status` 说明：`0`=发布成功，`1`=发布中，`2`=原创失败，`3`=常规失败

```js
async function queryPublishStatus(accessToken, publishId) {
  const url = `https://api.weixin.qq.com/cgi-bin/freepublish/querystatus?access_token=${accessToken}`;
  const response = await axios.post(url, { publish_id: publishId }, { proxy: false });
  return response.data;
  // 成功时返回：
  // {
  //   publish_status: 0,
  //   article_detail: { item: [{ idx: 1, article_url: '文章永久链接' }] }
  // }
}
```
